# SkillPath - Project Brief

## Vision

SkillPath is a **Career Operating System** that transforms the overwhelming journey of becoming a developer into a clear, trackable path. Just as Google Maps provides turn-by-turn navigation, SkillPath provides developers with:

- **Where they are** (current skill assessment)
- **Where they want to go** (target role selection)
- **The best route** (personalized learning roadmap)
- **Progress visibility** (weekly tracking and streaks)
- **Gap awareness** (skill gap analysis)
- **Opportunity matching** (job radar)

## Problem Statement

Every developer journey starts with confusion:

1. **Students don't know which skills actually matter** - Tutorials are scattered, no clear path
2. **Job descriptions are confusing** - Requirements are vague and inconsistent
3. **Portfolios don't match roles** - Projects built without strategy
4. **Progress is invisible** - No way to measure improvement week over week

## Solution

SkillPath provides a structured, gamified approach to career development:

1. **Skill Assessment** - Honest self-evaluation with level 0-4 scale
2. **Gap Analysis** - Deterministic matching against real job requirements
3. **AI-Powered Roadmaps** - Personalized learning paths generated from goals
4. **Weekly Sprints** - Maintainable weekly goals with streak tracking
5. **Portfolio Projects** - Strategic project recommendations that fill gaps
6. **Job Matching** - Real job listings with skill match scoring

## Target Users

- **Students** - CS/bootcamp students who want structured learning
- **Career Switchers** - Professionals transitioning to tech
- **Self-Taught Developers** - Those who learned without formal structure
- **Junior Developers** - Early career devs who want to level up

## Success Metrics

- User completes onboarding → Sets target role and initial skills
- User updates skills → Receives updated readiness score
- User generates roadmap → Gets structured weekly plan
- User completes sprint → Maintains streak and sees progress
- User matches with job → Sees skill gap and roadmap recommendation
